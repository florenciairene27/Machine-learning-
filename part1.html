Introduction

Abstract

This report aimed to build and choose the best machine learning for classifying defaulted loan based on 2012-2014 lending club data. The data split by 75-25(for training - testing). There are Five models built and tested, which are:

  1. KNN
  2. Naive bayes
  3. Logistic regression
  4. Decision tree C50
  5. Random Forest

Furthermore, each model parameter was tuned using 5 fold cross validation method, and AUC metric computed to choose the best model. Out of the five models, logistic regression performed the best.

Introduction

Lending club is a fintech company, with main service in consumers and commercial loan offering. In 2015 Lending Club became the largest peer to peer lending in the world.

The 2012-2014 from lending club data set contains 152 variables with over million records. Due to computing limitations, 10000 subset data are used and explored.

Finally, the question that will be answered from processing this data is: what is the best model to classify loan default?
